# Logo CFAO Mobility

Dépose ici le logo officiel sous le nom **`logo.png`** (ou `logo.svg`).

- Chemin attendu : `public/logo.png`
- Il est servi à l'URL `/logo.png` et apparaît automatiquement dans la barre
  latérale, l'écran de connexion et l'en-tête.
- Sur fond sombre (sidebar / login), il est posé sur une plaque blanche pour
  rester lisible.
- Format recommandé : PNG à fond transparent (ou SVG), hauteur ~32 px
  (s'adapte automatiquement).
- Tant qu'aucun fichier n'est présent, un logotype « cfao MOBILITY » fidèle à la
  marque est affiché.

> Si tu fournis un SVG, renomme `/logo.png` en `/logo.svg` dans
> `src/web/components/Logo.tsx` (une seule ligne).

## Comment déposer le fichier
- En local : copie ton image dans `public/logo.png` puis commit.
- Ou envoie-moi le fichier **en pièce jointe** (pas en image collée dans le
  chat) et je l'ajoute au dépôt.
