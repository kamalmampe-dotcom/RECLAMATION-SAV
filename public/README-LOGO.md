# Logo CFAO

Dépose ici le logo officiel sous le nom **`logo.svg`** (idéalement) ou `logo.png`.

- Chemin attendu : `public/logo.svg`
- Il sera servi à l'URL `/logo.svg` et apparaîtra automatiquement dans la barre
  latérale, l'écran de connexion et l'en-tête.
- Format recommandé : SVG sur fond transparent, hauteur ~36 px (s'adapte tout seul).
- Tant qu'aucun fichier n'est présent, l'application affiche un monogramme « CF ».

> Si tu fournis un `.png`, renomme l'`<img src="/logo.svg">` en `/logo.png` dans
> `src/web/components/Logo.tsx`, ou convertis simplement le fichier en SVG.
